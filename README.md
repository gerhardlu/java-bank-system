# Java Bank System

Sistema bancário em Java desenvolvido como projeto de estudo, simulando operações financeiras básicas como depósitos, saques, contas correntes e contas poupança.

## Sobre o projeto

Este projeto foi criado durante meus estudos de Engenharia de Software na FIAP, como forma de praticar Programação Orientada a Objetos, aplicada a um contexto próximo da minha área de interesse do setor bancário e fintech.

## Funcionalidades

- Criação de contas bancárias
- Depósito e saque com validações
- Conta Corrente: possui limite de cheque especial
- Conta Poupança: rende juros sobre o saldo
- Exibição de extrato/saldo da conta

## Conceitos praticados

- Classes e objetos
- Encapsulamento (atributos privados, acesso controlado via métodos)
- Herança (`ContaCorrente` e `ContaPoupanca` herdando de `Conta`)
- Sobrescrita de métodos (`@Override`)
- Validações e tratamento de regras de negócio

## Tecnologias

- Java 17
- IntelliJ IDEA
## Como executar

1. Clone o repositório:
```bash
   git clone https://github.com/gerhardlu/java-bank-system.git
```
2. Abra o projeto no IntelliJ IDEA (ou outra IDE Java de sua preferência)
3. Execute a classe `Main.java`
